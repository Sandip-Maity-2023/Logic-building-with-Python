# Food Delivery Analytics Platform

AI-powered analytics dashboard for Zomato, Swiggy, and Uber Eats datasets.

## Features

- Dataset upload
- Dashboard analytics
- Restaurant search
- Recommendation system
- Data visualization
- Flask backend
- Bootstrap frontend

## Installation

```bash
pip install -r requirements.txt
````

## Run

```bash
python app.py
```

Deploy using Render or Railway.
